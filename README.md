# Анализ доходов и расходов салона для кудрявых Waves

Учебный проект — Яндекс Практикум, курс «Аналитик данных», Спринт 1.

## Задача

Салон для кудрявых Waves работает на рынке 6 месяцев. Команда обратилась 
с запросом: найти точки для роста прибыли.

Анализ проводился по двум направлениям:
- промоакции — окупаются ли вложения в маркетинг
- товары и услуги — что пользуется наибольшим спросом

## Данные

Два листа Excel:
- `Export` — 5 356 транзакций за 7 месяцев (доходы и расходы)
- `promocodes` — справочник промокодов и маркетинговых кампаний

## Что сделала

**Предобработка**
- Удалены дублирующиеся строки
- Исправлены опечатки в категориях
- Нормализованы значения скидок
- Добавлены столбцы: вид транзакции, месяц, название кампании, 
  сумма скидки, итоговая стоимость

**Анализ**
- Отчёт о доходах и расходах по месяцам с детализацией по категориям
- Анализ рентабельности промокампаний: количество промокодов, 
  недополученная выручка, средний чек
- Анализ популярности товаров и услуг: количество продаж и выручка 
  по месяцам, чистая прибыль по товарам

## Выводы

Салон работает в плюс, выручка стабильна — рекорд в марте. 
Тренда на рост пока нет.

Кампания «Посты в Telegram» — наибольшая недополученная выручка 
(164 700 руб.) при минимальном среднем чеке (2 112 руб.).

Самая популярная услуга — стрижка и укладка. Устойчивый рост спроса 
на гель для волос из месяца в месяц.

## Рекомендации

- Пересмотреть кампанию «Посты в Telegram»
- Начать вести базу клиентов — без неё нельзя оценить эффективность акций
- Расширить ассортимент гелей и стайлинговых средств для кудрявых
- Развивать комбо-предложения по уходу за волосами
